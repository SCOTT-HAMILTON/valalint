WIP Vala Linter
===============

This is a work-in-progress Vala language linter for [Nuvola Apps project](https://github.com/tiliado/nuvolaruntime).

Installation
------------

Dependencies:

  - GNU Make
  - Vala 0.40.x
  - GLib/GIO
  - GNU diff

Build:

    make all
    make DESTDIR=... PREFIX=... install
